P4davan

Padavan RT-N56U firmware site by HuangYeWuDeng

  • NEWIFI-D1
  • NEWIFI-Y1
  • PSG1218-K2

This project is maintained by 80x86.io

固件特点

1.基于Padavan原版作少量修改. 跟随原版升级.

2.追求稳定，仅加入少量使用频率较高的插件，与的Padavan系统深度集成.

3.支持 在线升级 和 计划任务全自动升级

功能修改增强

  1. 完美适配D1, Y1, PSG1218-K2. D1 开启双核四线程，K2开启外置PA+LNA， Y1 开启外置LNA， led灯, 网口, 内置SD卡，U盘自动挂载，内置SMB, ftp, nfs, dlna 媒体服务器, itunes 媒体服务, Transmission 及 Aria2 下载器.

  2. 集成插件： 迅雷远程 adbyby KoolProxy SmartVPN SS kms JBLS ngrok shellinabox

  3. 增加qos模块支持

  4. 默认英文UI，增加中文语言可选.(插件没有做中文语言包)

  5. 集成dnspod, aliyun ddns, pubyun(3322) ddns, oray ddns, dynu ddns
